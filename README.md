## 🌟 About Spodude

**Spodude** is a personal music streaming application developed with ❤️ by a developer from **Tamil Nadu, India 🇮🇳**

The goal behind Spodude is simple:

> Create a smooth, beautiful and ad-free music experience using modern open-source technologies.

No unnecessary interruptions.
No clutter.
Just music.

---

## 🚀 Features

✨ **Ad-Free Experience**
Enjoy music without advertisements.

🎧 **Music Streaming Interface**
Clean and modern UI designed for music lovers.

⚡ **Fast Performance**
Built with Flutter for smooth cross-platform performance.

🌙 **Modern UI Design**
Beautiful animations, gradients and responsive layouts.

🔓 **Open Source Powered**
Built using community-driven technologies.

📱 **Mobile First Experience**
Optimized for Android devices.
